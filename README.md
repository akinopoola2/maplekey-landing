# MapleKey Landing Page

Coming Soon landing page for **MapleKey**, the Canadian property management SaaS platform.

Live at: [maplekey.app](https://maplekey.app)

## About MapleKey

MapleKey is a multi-tenant property management platform designed for Canadian property managers, landlords, and owner clients. It unifies three portals (tenant, manager, owner) into a single system with Canadian-specific features: Interac e-Transfer reconciliation, Manitoba RTA compliance, CRA T776 export, and bilingual support.

Founded by [Akindele Opoola](https://akinopoola.com), Royal LePage Top Producer and founder of Tekreal Canadian Inc.

## Tech Stack

- Static HTML (single file)
- Custom CSS with CSS variables
- Fraunces + Manrope via Google Fonts
- Vanilla JavaScript for email capture and scroll reveals
- Deployed on Vercel, DNS managed by Cloudflare

## Local Preview

Simply open `index.html` in any browser. No build step required.

## Deployment

This site auto-deploys to [maplekey.app](https://maplekey.app) via Vercel on every push to `main`.
